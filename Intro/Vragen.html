Antwoord: Het internet is een globaal (global) netwerk en het Web, ook wel World Wide Web genoemd is een collectie van informatie die toegangkelijk is via het internet. Internet is een infrastructuur en het Web is een service dat op het infrastructuur gegeven wordt. Wat is het verschil tussen HTML, CSS en JavaScript? HTML = HyperTextMarkupLanguage (opbouw/structuur van een website) CSS = Cascadin Style Sheet (het stylen van een website) JavaScript = is een programmeertaal (scripttaal dat gebruikt wordt voor het bouwen van kleine programma's) dat gebruikt wordt om een website levendig en interactief te maken. Wat is een HTML tag en wat is een HTML element? Hoe open en hoe sluit je een HTML tag? Een tag is een opdracht die door een browser kan worden geiterpreteerd. HTML gebruikt tags om bijvoorbeeld de positie en weergave van elementen op een pagina aan te geven.
